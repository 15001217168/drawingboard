## 简介 ##
这是一款给予h5 canvas的高级画板程序.
## 功能 ##
自由绘（颜色可切换）
<br />箭头
<br />圆（椭圆、正圆）
<br />三角形（等腰、直角）
<br />正方形
<br />文字
> 高级功能：
> <br /> 1.画直线时候，支持按住键盘Shift键，画垂直、水平线。
> <br /> 其他待续...
## 说明 ##
html+js+css，给予fabric.js开发完成的。
